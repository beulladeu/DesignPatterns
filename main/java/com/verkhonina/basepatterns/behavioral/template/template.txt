
Шаблонный метод
---------------

Поведенческий паттерн
    Определяет скелет алгоритма, перекладывая за некоторые его шаги на подклассы.
    Позволяет подклассам переопределять шаги алгоритма, не меняя его общей структуры.

    Абстракный класс определяет шаги алгоритма и содержит шаблонный метод, состоящий из вызовов этих шагов.
    Шаги могут содержать реализацию по умолчанию.
