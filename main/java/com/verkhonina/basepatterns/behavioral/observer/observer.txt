
Наблюдатель
-----------

Поведенческий паттерн


    механизм подписки
    позволяет одним объектам следить за другими и реагировать на их изменения

    Издатель содержит список подписчиков, методы подписки/отписки

    Подписчик содержит метод, которым пользуется издатель для отправки уведомления

    Клиент регистрирует подписчиков в издателе
